- strlrwr(string1) = converts a string to lower case
str (string) + lwr (lower)
- strupr(string1) =   converts a string to upper case

- strcat(string1, string2) =  appends string2 to end of string1
str (string) cat (concatenate)
strncat() = appends n characters from string2 to string1

strcpy(string1, string2) =  copy string2 to string1
strncpy(string1, string2, 4) =  copy n characters of string2 to string1

strset() -> set all characters of a string to a given characters
strnset -> sets first n characters of a string to a given characters
strrev -> reverses a string
strlen -> the length of of string

strcmp -> compare all characters
strncmp -> compare n characters
example: strncmp(string1, string2, 1) it will compare the first word on both string1 and string2
